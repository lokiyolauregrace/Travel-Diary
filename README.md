# 🌍 Travel Diary

Travel Diary is a Laravel web application where users can share travel experiences and discover places around the world ✈️

This project was made for the Backend Web course using Laravel 13.

---

# ✨ Features

- User registration and login
- Admin and normal user roles
- Public user profiles
- Create, edit and delete travel posts
- FAQ page
- Contact form
- Admin dashboard
- Responsive design

---

# 👤 User Profiles

Users can:
- Edit their profile
- Add a username
- Add a bio
- Add their birthday

Profiles are public and accessible for visitors.

---

# 📰 Travel Posts

Authenticated users can:
- Create travel posts
- Edit their own posts
- Delete posts
- View all travel posts

Each post contains:
- Title
- Country
- Content
- Publication date
- Image

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/lokiyolauregrace/Travel-Diary.git
```

Install dependencies:

```bash
composer install
npm install
```

Generate app key:

```bash
php artisan key:generate
```

Run migrations and seeders:

```bash
php artisan migrate:fresh --seed
```

Start the project:

```bash
php artisan serve
npm run dev
```
