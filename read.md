# 🌍 Travel Diary

A Laravel travel diary application where users can share their travel experiences around the world ✈️

## ✨ Features

- Authentication system
- Create, edit and delete travel posts
- FAQ page
- Admin role
- User profiles
- Comments system
- Contact messages
- Responsive girly UI 💖

## 🛠️ Built With

- Laravel 13
- PHP 8.4
- TailwindCSS
- SQLite
- Laravel Breeze
